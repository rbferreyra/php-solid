# PHP SOLID

Curso realizado na Shool of Net, explicando sobre os principios e fundamentos do [SOLID](https://pt.wikipedia.org/wiki/SOLID) utilizando a linguagem PHP.

## Requerimentos

- Composer
- PHP > 7

## Execução

Instalar as dependências do projeto

```bash
composer install
```

Realizar os testes

```bash
vendor/bin/phpunit
```
